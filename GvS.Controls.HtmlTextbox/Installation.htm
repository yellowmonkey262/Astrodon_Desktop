Installation - HtmlTextbox for Windows Forms

  1. Download the latest release
  2. Save the GvS.Controls.HtmlTextbox.Dll somewhere on your filesystem
  3. Place a link from the toolbox to the DLL
  4. You can now drag & drop controls on you forms
